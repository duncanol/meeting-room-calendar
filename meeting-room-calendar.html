Meeting Room bookings

The idea of this little application will be to allow an administrator to administer a list of Meeting Rooms and for normal users to be able to book those meeting rooms for slots of time.

Functionality will be tackled in the below order:

  1. Add new Meeting Room
  2. Edit Meeting Room
  3. Remove Meeting Room
  4. Make it look nice so far - introduce Bootstrap
  5. Reorder files and folders to make development nicer
  6. Book a slot for a Meeting Room
  7. Cancel a booking
  8. Try to make it look nice again!
  9. Adding User accounts and permissions
  10. Security - removing rapid prototyping features
  11. Deploying to node.js server
  12. Deploying to Meteor infrastructure

Current state of our application

So far all we have is:

  • this static piece of HTML being served by Meteor
  • a barebones javascript file with logging on both the client- and server-side
  • a default folder structure with room for improvement
